# My Portfolio

A modern, responsive portfolio website built with HTML, CSS, and JavaScript featuring:

- **Glassmorphism Design**: Beautiful glass effect with backdrop blur and transparency
- **Gradient Effects**: Colorful gradient backgrounds and text effects
- **Responsive Layout**: Mobile-first design that works on all devices
- **Interactive Elements**: Smooth animations, hover effects, and transitions
- **Project Showcase**: Attractive project cards with images and details
- **Skills Section**: Skill cards with progress bars and tech backgrounds
- **Statistics**: Animated counters for projects, clients, and experience
- **Contact Form**: Functional contact form for inquiries
- **Modern Navigation**: Top-right navigation pills with gradient styling

## Features

- Fixed side panel with "My Portfolio" label
- Split hero section with heading and illustration
- Navy blue background with glassmorphism effects
- Attractive skill cards with images and progress indicators
- Project cards with glassmorphism and gradient overlays
- Stats section with circular icons
- Responsive navigation at the top right
- Smooth scroll animations

## Technologies Used

- HTML5
- CSS3 (Glassmorphism, Gradients, Animations)
- Vanilla JavaScript (No frameworks)
- Google Fonts (Poppins, Inter)
- Unsplash for high-quality images

## How to Use

1. Open `index.html` in your browser
2. Navigate through sections using the top-right navigation pills
3. View projects by clicking "View Project" on project cards
4. Fill out the contact form to get in touch

## Customization

- Edit `index.html` to change content and structure
- Modify `style.css` for styling and colors
- Update `script.js` for interactive functionality
- Replace images in project/skill cards with your own
